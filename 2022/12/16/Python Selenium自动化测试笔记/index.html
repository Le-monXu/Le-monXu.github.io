Python Selenium自动化测试笔记

Lemon

Python Selenium自动化测试笔记，主要步骤流程：使用浏览器扩展插件Selenium IDE录制自动化流程，导出Python测试代码，编辑完善，完成自动化测试。

主要步骤

Selenium IDE插件录制自动化流程

  • 打开浏览器
  • 点击Selenium IDE插件
  • 输入项目名称，开始录制
  • 录制时尽量包括各种选项、意外
  • 录制结束，打开Selenium IDE插件点击停止录制
  • 调整每一条记录，最好使用XPATH记录
  • 选择Python导出，可选是否要注释

Python Selenium自动化流程编写

  • 打开导出的Python代码
  • 修改类名、方法名，增加测试入口代码，先执行setup_method方法，再执行业务代码，最后要执行teardown_method方法
  • 修改测试所用浏览器，Selenium IDE导出代码默认使用Chrome
  • 增加import代码，用于异常判断
import selenium.common.exceptions
  • 判断元素是否存在：使用XPATH查找，通过异常判断，元素无法交互、元素找不到均返回False
def is_element_exist(self,elementXpath):
        try:
            self.driver.find_element(By.XPATH, elementXpath)
            return True
        except selenium.common.exceptions.NoSuchElementException:
            return False
        except selenium.common.exceptions.ElementNotInteractableException:
            logger.info('ElementNotInteractableException')
            return False
  • 等待元素出现：基于判断元素是否存在修改，使用循环等待，可以加入超时错误
def wait_to_show(self,elementXpath):
        while True:
            if self.is_element_exist(elementXpath):
                break
            else:
                time.sleep(0.3)
                continue
  • 等待元素出现后点击：15秒未出现抛出TimeoutError
def wait_to_continue(self,elementXpath):
        tmp=0
        while True:
            if self.is_element_exist(elementXpath):
                # time.sleep(0.5)
                self.driver.find_element(By.XPATH, elementXpath).click()
                break
            elif tmp==50:
                logger.error(elementXpath+'---Timeout, raise TimeoutError')
                raise TimeoutError(elementXpath+'---Timeout, 元素实在是找不到了T_T')
                break
            else:
                time.sleep(0.3)
                tmp=tmp+1
                continue
